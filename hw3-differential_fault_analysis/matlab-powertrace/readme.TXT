(1) Power traces are stored in \texttt{PowerTrace.dat}.
There are 500 traces in this file; each trace contains has 30.00 points 
stored in 8 bit unsigned integer format.
There is no "\n" or any divider symbol in the file.

(2) The corresponding 500 plaintexts are saved in plaintext.dat file
Each plaintext consists of 16 numbers saved in 8 bit unsigned integer format.
There is no "\n" or any divider symbol in the file.

(3) The corresponding 500 plaintexts are stored in plaintext.txt file
Each plaintext consists of a 128-bit number in hexadecimal format, one number per line.
